# Attribution

This site is an independent teaching project.

It is inspired by:

- MIT 6.S081 / 6.828 Operating System Engineering.
- xv6, the teaching operating system from MIT PDOS.
- Local study notes and lab work developed in this workspace.

Not affiliated with MIT, PDOS, or the xv6 authors.

## What this project adds

- A concrete-state-first teaching format.
- Fill-in-the-blank lab derivations.
- Gotcha-first explanations.
- Local code audit tying each lesson to runnable xv6 branches and tests.

## Use short excerpts only

The public site should use short illustrative code snippets and diagrams, not
full copied solution files.
